# Weather Report

Script to get weather report in terminal

        USAGE:
            weather [FLAGS] [LOCATION]...
            
        FLAGS:
            -h   shows this help text'
            -f   shows full weather report for 3 days'
            -m   shows moon remort'
            -d   shows weather report for today'
            -l   Setting the location
